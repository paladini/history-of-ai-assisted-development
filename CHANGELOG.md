# Changelog

All notable changes to this project are documented here.

## [1.0.0] - 2026-06-30

### Added

- Initial release of The History of AI-Assisted Development site
- Interactive scroll-driven timeline with 45 milestones across 6 eras
- Category and era filters for timeline events
- 12 glossary pages (prompts, chat modes, MCP, A2A, agents, skills, multi-agent, AGENTS.md, vibe coding, context indexing, inline completion, trust paradox)
- Data dashboard with Recharts visualizations (adoption curve, trust paradox, use cases, agent adoption)
- Mermaid architecture diagrams (evolution ladder, MCP stack, MCP+A2A, agent loop)
- Stack Overflow survey data (2023–2025) in `data/surveys.json`
- Dark editorial theme with Instrument Serif + IBM Plex Sans typography
- GitHub Actions deploy workflow for GitHub Pages
- Open-source boilerplate: README, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY, SOURCES, CONTENT_GUIDE
- Issue templates for timeline events and bug reports
- OG meta tags and social preview image
- WCAG accessibility: skip links, keyboard navigation, reduced-motion support
